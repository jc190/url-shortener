# Url Shortener Microservice
Use a url query to return a JSON object containing the original url and a new shortened url.
****************
### __Example formats__
#### Shorten URL:
`https://urls.jamesc.me/new/http://www.google.com`
#### Use new URL:
`https://urls.jamesc.me/SJBYFNH9`
